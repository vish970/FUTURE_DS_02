# FUTURE_DS_02
#  Customer Retention & Churn Analysis

##  Overview
This project analyzes customer churn behavior in a subscription-based business using the Telco Customer Churn dataset.

The goal is to identify churn patterns, retention drivers, and provide actionable insights to reduce customer loss.

---

## Dataset
- Telco Customer Churn Dataset (Kaggle)

---

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Excel

---

##  Key Analysis
- Churn Rate Calculation
- Tenure Analysis
- Customer Segmentation
- Monthly Charges vs Churn
- Contract Type Analysis

---

##  Key Insights
- High churn occurs in early customer lifecycle
- Month-to-month contracts have highest churn
- Higher monthly charges increase churn probability
- Long-term customers show strong retention

---

##  Recommendations
- Improve onboarding for new users
- Offer long-term contract incentives
- Target high-risk customers with retention strategies
- Introduce loyalty programs
